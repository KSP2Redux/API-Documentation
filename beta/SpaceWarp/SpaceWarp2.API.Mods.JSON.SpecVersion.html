    Specification version 1.3 (SpaceWarp 1.3.0 - 1.4.3) - adds back the ModID field, but enforces that it is the same as the BepInEx plugin GUID if there is a BepInEx plugin attached to the swinfo, since w/ version 1.3 there does not have to be a plugin.
