    AddFlowActionToGameLoadAfter(IFlowAction, string)

    Add a FlowAction to the Game's load sequence.

    The action will be run after the first FlowAction with a name equal to referenceAction. If referenceAction is null, the action will be the first action run.

    The FlowActions that occur in this sequence by default are: (Updated for KSP 0.1.1.0)

    1. "Creating Splash Screens Prefab"
    2. "Set Loading Optimizations"
    3. "Initialize Addressables system"
    4. "Creating Game Instance"
    5. "Initializing Game Instance"
    6. "Creating Graphics Manager"
    7. "Creating Main Menu"
    8. "Load UI Background Scenes"
    9. "Parsing Loading Screens"
    10. "Set Loading Optimizations"
    11. "loading addressable localizations" (Added by SpaceWarp)
