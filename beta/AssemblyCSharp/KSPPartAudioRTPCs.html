    Namespace KSPPartAudioRTPCs

    Classes

    KSPPartAudioRTPCBase

    A base class for KSP part audio RTPC entries that update real-time parameter control values on audio game objects.

    PartAngularVelocityRTPC

    RTPC entry that drives the k_angular_velocity audio parameter using the angular velocity magnitude of the owning part's rigidbody.

    PartAtmDensityRTPC

    Represents the atmospheric density RTPC for part audio, setting the audio parameter from the part's current atmospheric density.

    PartDragCoefficientRTPC

    Represents a part audio RTPC handler that drives the drag-coefficient runtime parameter using the part model's drag coefficient.

    PartDynamicPressureRTPC

    RTPC entry that updates the dynamic pressure real-time parameter control value on a part's audio game object.

    PartGForceRTPC

    Part audio RTPC entry that drives the g-force real-time parameter control value from the vessel's current g-force.

    PartJointTorqueRTPC

    Represents a KSPPartAudioRTPCBase implementation that drives the part stress audio parameter using the maximum joint torque of the owning part.

    PartMachNumberRTPC

    Part audio RTPC that reads the current vessel Mach number and applies it to the part audio system.

    PartMassRTPC

    Represents a part audio RTPC entry that drives the part mass real-time parameter control value from the part's physics mass.

    PartSpeedRTPC

    Represents a KSPPartAudioRTPCBase implementation that updates the part speed RTPC with the vessel's surface speed magnitude.

    PartStaticPressureRTPC

    RTPC entry that updates the static pressure real-time parameter control value on a part audio game object using the part's static pressure in kPa.

    PartTemperatureRTPC

    RTPC entry that drives the part temperature real-time parameter control from the vessel's external temperature.

    PartUnderWaterRTPC

    Represents a KSP part audio RTPC entry that controls the k_part_under_water parameter, setting it to 1 when the part model is submerged and 0 otherwise.

    PartVelocityMagnitudeRTPC

    Represents a part audio RTPC entry that updates the velocity magnitude parameter using the part's speed relative to the SOI body frame.

    TemperatureRatioRTPC

    RTPC entry that drives the part temperature ratio audio parameter from the heat indicator tracker's temperature ratio value.

    VesselAltitudeRTPC

    Represents a KSP part audio RTPC entry that drives the vessel altitude real-time parameter control value on part audio game objects.

    VesselAtmDensityRTPC

    Represents a KSPPartAudioRTPCBase that updates the vessel atmospheric density RTPC parameter on part audio game objects.

    VesselHeightFromGroundRTPC

    Represents a part audio RTPC entry that drives the vessel height-from-ground parameter using the active vessel's altitude above terrain.

    VesselStaticPressureRTPC

    KSP part audio RTPC entry that drives the vessel static pressure parameter with the vessel's current static pressure in kPa.
