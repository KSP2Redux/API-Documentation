    Syntax
    public abstract class IndexedListUserData : JsonUserData
    Remarks

    Subclasses provide an item -> name function via Name(JToken) (and optionally a custom item conversion via Convert(JToken)), and the base class maintains parallel Indices and Conversions caches that translate string keys to array positions and to wrapped item values. Lua scripts then treat the array as if it were a Lua table keyed by item name -- string indexing, Remove(string), and iteration via Pairs() all operate on names rather than raw JSON object keys. Mutations (Append(DynValue), Insert(int, DynValue), Remove(int), Clear()) update the caches in step. String-indexed assignment is intentionally unsupported and always throws -- callers must use the explicit add/remove methods so the name-to-index mapping stays consistent.
