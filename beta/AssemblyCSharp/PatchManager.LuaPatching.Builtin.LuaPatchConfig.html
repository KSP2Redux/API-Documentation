    Class LuaPatchConfig

    Redux configs exposed to patch manager. Bindings declared through this object are recorded in ConfigReplay so changes to their values invalidate the patch cache.

    Color(ScriptExecutionContext, string, string, Table, string)

    Binds a color config for use in the patching engine. Any changes to the value will trigger an invalidation of the patch cache on the next launch. Colors are passed and returned as Lua tables; either keyed ({r=, g=, b=, a=}) or indexed ({1, 0.5, 0, 1}) input is accepted, and output is keyed.
