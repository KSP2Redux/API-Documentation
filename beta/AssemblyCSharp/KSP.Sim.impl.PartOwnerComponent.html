    DetachPart(PartComponent, Guid?, bool, bool)

    Recursively remove everything attached to part from the PartOwnerComponent that currently owns it, and move them to a new vessel. Various data from the previous vessel is carried over, such as science data.

    If forceRemove is false and the part is still attached to the vessel, then this has the effect of destroying the original VesselComponent and creating a similar one rooted at part.

    Declaration
    public void DetachPart(PartComponent part, Guid? promotionGuid = null, bool forceRemove = false, bool ignoreChildrenOfCompoundParts = false)
    Parameters
    Type Name Description
    PartComponent part

    The part to remove from the vessel.
    Guid? promotionGuid

    GUID of a subvessel record on the part's current vessel to use to construct the new vessel.
    bool forceRemove

    If this method should sever all of the current attachments to part
    bool ignoreChildrenOfCompoundParts

    If the process of dermining what is connected to part should ignore attachments flagged with IsCompoundJoint. (Struts, joint refinorcement, fuel tubes, etc.
