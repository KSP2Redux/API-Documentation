    Namespace KSP.FX.Timeline

    Classes

    CFXSCueConsumerComponent

    MonoBehaviour component that subscribes to audio, VFX, and stop cue messages for a named FX group and triggers the corresponding contextual effects in the timeline FX system.

    ContextualFXSystemBehavior

    Represents a PlayableBehaviour that activates and deactivates contextual FX events, publishing audio, VFX, or general messages via the message center.

    ContextualFXSystemData

    Represents a Timeline clip asset that holds configuration data for a contextual FX system.

    ContextualFXSystemMixer

    PlayableBehaviour mixer that activates and deactivates ContextualFXSystemBehavior clips on a Unity Timeline track.

    ContextualFXSystemTrack

    A Timeline track that binds to a SequenceControllerComponent and mixes ContextualFXSystemData clips.

    EffectAtLocationBehavior

    Represents a playable behavior that spawns a VFX effect at a tagged scene location during timeline playback.

    EffectAtLocationData

    Represents a Timeline clip asset that creates an EffectAtLocationBehavior playable.

    EffectAtLocationMixer

    Represents a PlayableBehaviour mixer that activates and deactivates location-based effects as timeline clips start and end.

    EffectAtLocationTrack

    Represents a Timeline track that plays effects at world-space locations using EffectAtLocationData clips bound to a TimelineAdapter.

    PartAudioEventBehavior

    Represents a timeline playable behaviour that fires a part audio event on activation.

    PartAudioEventData

    Represents a Unity Timeline clip asset that drives part audio events via a PartAudioEventBehavior.

    PartAudioEventMixer

    Represents a PlayableBehaviour mixer that activates or deactivates part audio event clips based on their play state during timeline playback.

    PartAudioEventTrack

    Represents a Unity Timeline track that drives part audio events.

    PrelaunchTimelineController

    Manages the prelaunch and launch timeline visual effects for a vessel on the launchpad, coordinating TimelineAdapter instances in response to vessel creation and launch events.

    SequenceControllerComponent

    Represents a KerbalMonoBehaviour component that manages playback of named Unity Timeline sequences on a PlayableDirector, adjusting speed in response to time-warp changes.

    SequenceControllerComponent.NamedTimeline

    Represents a named PlayableAsset entry used for timeline lookup by name in a SequenceControllerComponent.

    TimelineAdapter

    Represents a KerbalMonoBehaviour that adapts a Unity Timeline PlayableDirector for KSP FX sequences, coordinating time-warp speed scaling, VFX spawning, audio events, and object-proxy resolution.
