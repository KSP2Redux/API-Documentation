    Namespace PatchManager.Core.Cache.Json

    Classes

    CacheEntry

    Represents an entry in the cache for a specific asset label.

    ConfigReplayConstraint

    Persisted form of an IValueConstraint for replay. Discriminated by Kind: range uses Min / Max, list uses Values.

    ConfigReplayEntry

    Persisted record of a single Config: binding declared by a Lua patch. Stored under Entries and replayed by ConfigReplay so the binding survives hot-cache launches that skip running the Lua scripts.

    ConfigReplaySlice

    Per-mod replay slice carrying identity (descriptor versus standalone, plus the .lua path for standalones) and the bindings the mod's patches declared. Stored as SerializedConfigs values so ConfigReplay can decide at replay time whether the mod still exists before re-binding its entries.

    Inventory

    Catalog of all patched labels and their respective archives, persisted to disk between runs.

    PatchHashes

    Contains hashes of the cache.
