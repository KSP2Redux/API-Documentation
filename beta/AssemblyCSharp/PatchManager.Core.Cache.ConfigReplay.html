    Class ConfigReplay

    Tracks Config: bindings declared by Lua patches so they can be replayed on launches where the patch cache is hot and the scripts therefore do not re-run. Replay re-creates the bindings against the appropriate IConfigFile so the settings UI is populated, and compares the current value to the value the patch ran with to flag stale caches that need invalidating.

    ReplayAll()

    Replays every recorded binding against its mod's config file. Re-binds the entry so it shows up in the settings UI even on hot-cache launches, and flags ChangedKeys for any entry whose current value differs from the recorded ran-with.

    Declaration
    public static void ReplayAll()
    Remarks

    Should run early at startup, after SpaceWarp has populated PluginList but before deciding whether to use the patch cache. Slices for standalone patches whose .lua file no longer exists, or descriptor mods that are not loaded this run, are pruned from the inventory: the recorded ran-with no longer reflects reality. User-set values stay in the underlying .json config files and are re-bound automatically when the mod returns and its patches run again.
