    Namespace KSP.Game.Flow

    Classes

    FlowAction

    A base class for a named, timed action within a game flow sequence, executed asynchronously with resolve and reject callbacks.

    FreezeProceduralPartDefinitionDatabaseFlowAction

    Flow action that freezes and caches the procedural part definition database, preventing new definitions from being added.

    FreezeResourceDefinitionDatabaseFlowAction

    Flow action that freezes the resource definition database, preventing new resource definitions from being added.

    GenericFlowAction

    A FlowAction implementation that executes a caller-supplied delegate as the action's work.

    LoadAssetFlowAction<T>

    A FlowAction that loads a Unity asset of type T from an AssetProvider using a string key.

    LoadAssetReferenceFlowAction<T>

    FlowAction that loads an UnityEngine.AddressableAssets.AssetReferenceT<TObject> using an AssetProvider and delivers the result through a callback.

    LoadAssetReferencesFlowAction<T>

    FlowAction that loads a list of Addressable asset references of type T via an AssetProvider.

    LoadAssetsFlowAction<T>

    FlowAction that loads a batch of assets of type T from an AssetProvider, invoking a per-asset callback and a completion callback on finish.

    LoadStartingCelestialBodyFlowAction

    A FlowAction that loads the starting celestial body's space view during game load, waiting for its local-space or scaled-space representation to finish spawning.

    PopulateProceduralPartDatabaseFlowAction

    Flow action that loads procedural part assets and populates the procedural part definition database.

    PopulateResourceDefinitionDatabaseFlowAction

    FlowAction that loads resource definition assets and populates the game's resource definition database.

    ReduxFlowAction

    Wraps an IFlowAction as a KSP FlowAction, delegating name, description, and execution to the underlying Redux action.

    SaveGameMetadataLoader

    Manages a collection of SavedGameMetaDataLoaderInstance instances that load metadata from saved game files.

    SaveGameThumbnailCreator

    Creates thumbnail image files for save game entries by asynchronously capturing the screen and encoding the result to disk.

    SavedGameMetaDataLoaderInstance

    Represents a single asynchronous instance for loading saved-game metadata from disk, invoking a callback when the load completes or fails.

    SequentialFlow

    Represents a sequential execution pipeline that runs a list of FlowAction items one at a time, tracking flow state, progress, and errors.
