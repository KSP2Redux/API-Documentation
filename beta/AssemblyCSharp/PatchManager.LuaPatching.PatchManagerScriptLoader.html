    Remarks

    Module resolution understands the modGuid:moduleName namespacing convention so a script can require library files (filenames prefixed with _) from another mod's folder. A bare require("modname") looks inside the host mod's own folder, identified by the script's ModId global. File resolution joins relative paths against the script's Location global.
