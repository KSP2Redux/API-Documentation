    Class CommandBinding

    Convenience binding for Button.clicked -> ICommand. Wires the button's clicked event and enable state to a parameterless command on the data source.

    UXML usage:

    <ui:Button name="buy-btn" text="Buy">
    <Bindings>
        <CommandBinding property="text" data-source-path="BuyCommand" />
    </Bindings>
</ui:Button>

    The "property" attribute is required by the infrastructure but unused - set it to any valid property (by convention "name"). For non-Button elements, use EventBinding.
