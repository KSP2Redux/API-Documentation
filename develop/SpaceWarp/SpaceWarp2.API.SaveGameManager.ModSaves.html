    RegisterSaveLoadGameData<T>(string, Action<T>?, Action<T>?, Func<T, T>?, T?, PersistenceKind)

    Registers your mod data for saving and loading events.

    Declaration
    public static T RegisterSaveLoadGameData<T>(string modGuid, Action<T>? onSave = null, Action<T>? onLoad = null, Func<T, T>? onNew = null, T? saveData = null, PersistenceKind persistenceKind = PersistenceKind.PerSave) where T : class
    Parameters
    Type Name Description
    string modGuid

    Your mod GUID. Or, technically, any kind of string can be passed here, but what is mandatory is that it's unique compared to what other mods will use.
    Action<T> onSave

    Function that will execute when a SAVE event is triggered. Defaults to null or no callback.
    Action<T> onLoad

    Function that will execute when a LOAD event is triggered. Defaults to null or no callback.
    Func<T, T> onNew

    Function that will execute when a LOAD event happens where there was not data prior in the save or campaign for this mod. What is passed in is the old data, and what is returned becomes the new data.
    T saveData

    Your object that will be saved to a save file during a save event and that will be updated when a load event pulls new data. Ensure that a new instance of this object is NOT created after registration.
    PersistenceKind persistenceKind

    How the save data is persisted, per save or per campaign, your mod can register save data in one of each of these slots
    Returns
    Type Description
    T

    T saveData object you passed as a parameter, or a default instance of object T if you didn't pass anything
    Type Parameters
    Name Description
    T

    Any object
