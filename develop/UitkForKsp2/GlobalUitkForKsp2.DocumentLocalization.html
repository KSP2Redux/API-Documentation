    Class DocumentLocalization

    A component which automatically localizes all elements in a document. Only elements with a string property "text" whose value is a localization key starting with '#' will be localized.

    The LocalizationManager.OnLocalizeEvent is handled to automatically update the localization of all registered elements when the game language is changed.
