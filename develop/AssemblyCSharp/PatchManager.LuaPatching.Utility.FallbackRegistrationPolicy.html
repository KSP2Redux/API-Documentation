    Class FallbackRegistrationPolicy

    MoonSharp MoonSharp.Interpreter.Interop.RegistrationPolicies.IRegistrationPolicy that wraps every UserData type with a string indexer in an IndexerFallbackDescriptor and disallows automatic type registration.

    public class FallbackRegistrationPolicy
    Remarks

    Installed as the global UserData.RegistrationPolicy by Universe's static constructor, so any JsonUserData subclass registered later automatically gets member-lookup-then-indexer dispatch without having to opt in per-type. Auto-registration is disabled to keep the Lua-exposed type set explicit; every type must carry MoonSharp.Interpreter.MoonSharpUserDataAttribute to be visible from scripts.
