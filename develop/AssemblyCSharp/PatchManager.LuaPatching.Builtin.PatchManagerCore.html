    ImplicitStage(Script, string)

    Creates a named stage that runs after the most recent implicit stage from the same host mod.

    Declaration
    public Stage ImplicitStage(Script context, string name)
    Parameters
    Type Name Description
    Script context

    The host Lua script; its ModId global is used to scope the implicit chain.
    string name

    The stage name (registered as "modId:name").
    Returns
    Type Description
    Stage

    The created stage, suitable for chaining (for example Before(CallbackArguments) / After(CallbackArguments)).
    Remarks

    Use this when stages within a mod should run in declaration order. The first ImplicitStage in a mod runs after the mod's own base stage, and each subsequent one runs after its predecessor. The mod's __post stage is also updated to run after the new stage so the mod's post stage stays at the tail of the mod's chain. The stage is registered under "modId:name"; references via OnStage(string), Before(CallbackArguments), and After(CallbackArguments) use that fully-qualified name.
