    Class JsonUserData

    Lua-facing wrapper exposing a Newtonsoft Newtonsoft.Json.Linq.JToken as a table-like UserData.

public class JsonUserData
    Remarks

    Patch scripts read and mutate JSON assets through this type: object keys become string indices, array elements become 1-indexed integer indices (matching Lua convention), and iteration is exposed through the __pairs / __ipairs metamethods on Pairs(). Reads against the wrong token type are non-fatal -- they log a debug message and return MoonSharp.Interpreter.DynValue.Nil or an empty result. Writes against the wrong token type throw, since silently accepting a write would lose data. Subclasses (IndexedListUserData, ExtensibleJsonUserData) override these members to layer name-indexed lookup or virtual properties on top of the underlying JSON.

    Remarks

    Lua numbers become integers when they have no fractional part. Lua tables become JSON arrays when their integer length is non-zero, JSON objects otherwise.
