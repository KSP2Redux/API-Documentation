    Class GenericLuaModule

    Lua submodule exposed as PM.JSON, providing label-untyped patch and asset-creation helpers backed by the generic JSON converter.

    Inheritance
    object
    GenericLuaModule
    Namespace: PatchManager.Generic
    Assembly: Assembly-CSharp.dll
    Syntax
    [PatchManagerModule("JSON")]
[MoonSharpUserData]
public class GenericLuaModule
    Remarks

    Use this when patching addressables labels that do not have a domain-specific submodule, or to bypass the typed UserData wrappers and operate directly on raw JsonUserData.
